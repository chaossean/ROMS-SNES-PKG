# ROMS-SNES-PKG
¡Bienvenidos al almacén de ROMs de Super Nintendo de Chaos Sean!

En este repositorio encontrarás mis archivos PKG personalizados listos para instalar y disfrutar en tu PlayStation 3.

Para instalarlos, solo necesitas descargar el archivo PKG de tu juego favorito, pasarlo a una memoria USB y usar la opción "Install Package Files" en el menú de la consola.
Tambien pudieses pasarlo desde una PC a tu PS3 usando el programa FILEZILLA y conectarte a la PS3 por medio de la direccion IP, deben estar en la misma red ambos (PS3 y PC).

¡Que te diviertas reviviendo estos clásicos!
